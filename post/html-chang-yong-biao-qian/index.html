HTML常用标签

a 标签的用法有哪些?

  • 属性
    • href
      1. 网址 : //google.com
      2. 路径:./index.html
      3. 伪协议:JavaScript:; 此协议主要运用于当前超链接不想做其他事的时候使用
      4. mailto✉️
      5. tel☎️
      6. ID
        1. href 的值等于点击当前超链接要跳转过去元素的ID
    • target _blank 表示新建窗口打开 self 表示在当前页面打开
    • rel=noopener
      • 如果你需要用 a 标签打开一个标签页时，你会使用 target='_blank' 这个属性，此时你需要添加 rel='noreferrer noopener'
      • 这个是为了解决安全性问题.

img标签的用法?

  • 属性
  1. src 可以是文件路径 也可以是网络地址 通常网络地址前面不加协议 因为一旦写错地址就会引发很多bug,所以最优写法比如://google.com
    1. //google.com 这样写的好处就是不管 // 前面是什么协议,最终请求的时候浏览器会自己寻找最终这个地址的协议头
  2. alt 当图片加载不出来的时候提示用户此图片信息
  3. width / height: 设置宽高,一般适合宽度,高度让图片自适应,同时设置会导致图片变形,其实你知道图片具体宽高设置也是可以的,保险起见只设置宽度.

  • 事件

    1. onload: 加载成功需要做出的操作
    2. onerror:加载失败需要做出的操作

  • 响应式

    • Max-width:100%; 设置此CSS就是为了让图片在所有设备宽度都能够自动适应当前设备.

    ~~可替换元素~~ 当图片加载失败的时候可以使用JavaScript中 监听 img 事件 onerror 来操作.

table 标签的用法?

绘制表格使用；如果是做后台管理的用户列表等类似 Excel 功能，这个标签是最佳的选择

table 定义HTML中的表格元素
table标签中一般会包括多个th,tr,td
thead 表格头部
tbody 表格身体
tfoot 表格脚部
th 头
tr 行
td 列
