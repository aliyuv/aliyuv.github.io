HTML 起手应该写什么?

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
    <h1> hello world </h1>
</body>

常用的章节标签有哪些?分别是什么意思呢?:

  1. h1——h6 :标题
  2. section:章节
  3. article:文章
  4. p:段落
  5. header:头部
  6. footer:脚部
  7. main:主要内容
  8. div:旁支内容

HTML全局属性有哪些?

  1. class -> 类名
  2. contenteditalbe -> 枚举属性,表示元素是否可被用户编辑
  3. hidden -> 布尔属性表示该元素尚未或者不在相关,列如,它可用于隐藏在登录过程完成之前无法使用的页面元素,浏览器不会呈现此元素.但是不得使用此属性隐藏合法显示的内容
  4. id -> 定义唯一标识符(ID),该标识在整个文档中必须是唯一的. 类似每个人的身份证ID
  5. style -> 应用于元素的CSS样式声明
  6. tabindex -> 指示元素是否可以获取输入焦点(可聚焦) -> 分别有几个值:
    1. 负值表示该元素是可以聚焦的,但不应该通过顺序键盘导航到达
    2. 0表示元素应该通过顺序键盘导航可聚焦和可到达,但其相对顺序由平台约定定义
    3. 正值意味着元素应该可以通过顺序键盘导航进行聚焦和访问;元素聚焦的顺序是tabindex的增加值。如果多个元素共享相同的tabindex，则它们的相对顺序遵循它们在文档中的相对位置。
  7. title -> 表示与其所属元素相关信息的文本

常用的内容标签有哪些，分别是什么意思?

  1. ol li -> 表示有序的列表和列表中的每一个子列表
  2. ul li -> 表示无序的列表和列表中的每一个子列表
  3. dl dt dd -> 描述列表+被描述对象+描述的具体内容
  4. per -> 如果想在HTML中保留空格以及TAB等格式就可以使用此标签
  5. code -> 等宽的意思. 可以用per包裹code ,可以展示代码以及命令行相关的bash
  6. hr -> 水平分割线
  7. br -> 换行换行缩进
  8. em -> 语义化:指强调的语气
  9. strong -> 语义化: 本身就是很重要,默认样式是文本加粗.
  10. quote -> 内联的引用
  11. blockquote -> 块级的引用 (会换行缩进)
